Kod Açıklaması: Kod, kullanıcının konsoldan girdiği bilgilerden oluşan basit bir web sayfası hazırlar. Kullanıcıdan belirli sorulara aldığı yanıtlardan sonra bu yanıtları içeren web sayfasını oluşturmak için gereken HTML kodlamasını yazan bir index.html dosyası oluşturur. Kullanıcı ilk önce isim girişini yapar ve girdi, ad isimli değişkende tutulur. Daha sonra alınan ders sayısı kadar dönen bir döngü içerisinde alınan ders isimleri Enter ayrımıyla tek tek kullanıcıdan alınır. Girilen ders isimleri dersler isimli listede tutulur. En sonunda kullanıcının biyografisi alınır ve biyografi isimli değişkende tutulur. 

HTML dosyasının içerikleri html_parça isimli listede tutulur. İlk kısımda <head> etiketi içerisinde sayfanın başlığı ve CSS tasarımı bulunur. <body> etiketi içerisinde ise konsoldan girilen girdilerin tutulduğu değişkenlerin yazdırıldığı headingler ve listeler bulunur. Alınan dersleri içeren liste elemanları indisleri ile sırasıyla HTML'in sıralı liste etiketine yerleşir.

En sonda ise html_parça listesinin elemanları html değişkeninde birleştirilir. Write metoduyla açıan index.html dosyası oluşturulur ve içeriğine html değişkeni yazdırılır.

Kodun çalışması için python 3'ün kurulu olması gereklidir. Terminalden src dosyasına ulaşarak python3 src.py komutuyla çalıştırılabilir.
